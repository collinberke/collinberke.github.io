How much pressure does a timeout add to a fieldgoal?

football
kicking
Author

Wes Erickson

Published

November 4, 2025

Many coaches call a timeout right before a kicker kicks a field-goal. Does this really do anything?

In the NFL many games come down to the last minute and a team could only need a field goal. At this point many coaches still have a timeout. So how does this affect a kicker can some do better and some do worse? Does this distance matter on the kick to determine how well a kicker might do?

We need to find the data for fieldgoals and search through to see if there was a timeout called before. So we are going to use nflfastR package so we can go through the NFL play-by-play searching for the results that we want.

# CHUNK 2: Data Cleaning and Variable Creation
pbp_analysis <- pbp_data |>
  arrange(game_id, play_id) |>
  mutate(prev_play_is_timeout = lag(timeout, default = 0)) |>
  mutate(
    prev_timeout_team = lag(timeout_team, default = "NO_TEAM"),
    def_team_on_fga = defteam 
  ) 

fga_analysis <- pbp_analysis |>
  filter(play_type == "field_goal") |>
  filter(!is.na(field_goal_result)) |>
  mutate(
    fg_made = case_when(
      field_goal_result == "made" ~ 1,
      field_goal_result %in% c("missed", "blocked") ~ 0,
      TRUE ~ NA_real_ 
    ),
    iced_kicker = as.integer(
      prev_play_is_timeout == 1 & prev_timeout_team == def_team_on_fga
    )
  ) |>
  filter(!is.na(fg_made)) |>
  filter(!is.na(kicker_player_name)) |>
  select(kicker_player_name, kick_distance, field_goal_result, fg_made, iced_kicker, game_id, play_id)

# REVISED CHUNK 3 Data Preparation for Visualization
distance_rates <- fga_analysis |>
  filter(kick_distance >= 15, kick_distance <= 70) |>
  mutate(iced_kicker = factor(iced_kicker)) |> 
  mutate(
    distance_bin = cut(kick_distance, 
                       breaks = c(19.0, 29.5, 39.5, 49.5, 59.5, 70),
                       labels = c("20-29 yds", "30-39 yds", "40-49 yds", "50-59 yds", "60+ yds"),
                       right = TRUE, 
                       include.lowest = TRUE)
  ) |>
  filter(!is.na(distance_bin)) |>
  group_by(distance_bin, iced_kicker) |>
  summarise(
    success_rate = mean(fg_made),
    count = n(),
    .groups = 'drop'
  ) |>
  complete(distance_bin, iced_kicker, fill = list(success_rate = NA, count = 0)) |>
  filter(!is.na(success_rate) | count > 0)
Code 
# CHUNK: 3.5 - Data for gt Table
accuracy_table_data <- distance_rates |>
  filter(!is.na(iced_kicker)) |>
  select(distance_bin, iced_kicker, success_rate, count) |>
  pivot_wider(
    names_from = iced_kicker,
    values_from = c(success_rate, count)
  ) |>
  rename(
    `Non_Iced_Rate` = `success_rate_0`,
    `Iced_Rate` = `success_rate_1`,
    `Non_Iced_Count` = `count_0`,
    `Iced_Count` = `count_1`
  ) |>
  mutate(
    `Accuracy_Change` = Iced_Rate - Non_Iced_Rate
  ) |>
  select(
    `Distance_Yds` = distance_bin,
    `Non_Iced_Rate`, `Iced_Rate`, `Accuracy_Change`,
    `Non_Iced_Count`, `Iced_Count`
  ) |>
  filter(!is.na(Iced_Rate), !is.na(Non_Iced_Rate))

accuracy_table_final <- accuracy_table_data |>
  mutate(
    Non_Iced_Rate = scales::percent(Non_Iced_Rate, accuracy = 0.1),
    Iced_Rate = scales::percent(Iced_Rate, accuracy = 0.1),
    Accuracy_Change_Label = scales::percent(`Accuracy_Change`, accuracy = 0.1, sign = TRUE)
  )

## CHUNK 4: Beeswarm Plot Data Preparation 
MIN_NON_ICED <- 10
MIN_ICED <- 5

kicker_beeswarm_data <- fga_analysis |>
  filter(!is.na(kicker_player_name), !is.na(iced_kicker)) |>
  mutate(iced_kicker = as.character(iced_kicker)) |>
  group_by(kicker_player_name, iced_kicker) |>
  summarise(
    success_rate = mean(fg_made, na.rm = TRUE),
    count = n(),
    .groups = 'drop'
  ) |>
  group_by(kicker_player_name) |>
  filter(
    sum(iced_kicker == "0" & count >= MIN_NON_ICED) > 0,
    sum(iced_kicker == "1" & count >= MIN_ICED) > 0
  ) |>
  ungroup() |>
  mutate(
    success_per_100 = success_rate * 100
  ) |>
  mutate(
    kicker_status = factor(iced_kicker, 
                           levels = c("0", "1"),
                           labels = c("Non-Iced Attempts", "Iced Attempts"))
  )

Final chart that we are going to look at a bee swarm chart. When we look at this we are able to see that the pressure effects every kicker differently. When you look at the graph you are able to see guys like Tyler Bass and Daniel Carlson who kick at a 60% rate when a timeout is called on them.

In the end we find that the longer the field goal gets the more the timeout seems to cause them to miss. We also find that some people take pressure better then others and finally that we need way more data if we really want a definitive answer on if “icing” the kicker does anything.
